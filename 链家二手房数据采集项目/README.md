# 链家二手房数据采集与分析项目 📊
一个完整的Python爬虫+数据分析项目，从链家采集东莞二手房数据，完成清洗、统计与可视化，适合新手学习全流程开发。

---

## 项目简介
本项目以链家网东莞二手房数据为目标，通过**Playwright同步爬虫**采集房源信息，再用**Pandas**完成数据清洗与统计分析，最后用**Matplotlib**生成可视化图表，完整覆盖了「数据采集→处理→分析→可视化」的项目开发流程。

---

## 项目结构
```text
链家二手房数据采集项目/
├── main.py                # 爬虫主程序，负责页面爬取与数据解析
├── analysis/              # 数据分析与可视化模块
│   ├── analysis.py        # 核心统计函数（房价、区域、面积分析）
│   └── charts.py          # 可视化生成函数（柱状图/饼图/折线图）
├── result.xlsx            # 采集并清洗后的二手房数据文件
└── README.md              # 项目说明文档
```

## 快速开始
### 1. 克隆项目到本地
```bash
git clone https://github.com/yjfyyds2333/python_script_projects.git
cd python_script_projects/链家二手房数据采集项目
```

### 2. 运行数据采集（爬虫模块）
直接运行`main.py`即可启动爬虫，默认爬取东莞二手房数据，支持多页翻页：
```bash
python main.py
```
- 爬虫会自动处理页面加载、网络请求等待，内置随机延迟模拟真人行为
- 遇到人机验证或登录跳转时，可在弹出的浏览器窗口手动完成，程序会自动继续
- 爬取完成后，数据会自动保存为`result.xlsx`文件

### 3. 运行数据分析与可视化
爬虫结束后，执行分析模块生成统计结果与图表：
```python
# 调用分析与可视化函数（可直接在main.py中添加调用）
from analysis.analysis import remen_area
from analysis.charts import zhu_chart, bin_chart, line_chart

# 1. 生成区域平均房价柱状图
zhu_chart()
# 2. 生成热门区域房源占比饼图
bin_chart()
# 3. 生成房源面积分布折线图
line_chart()
```
所有图表会自动保存为PNG文件，可直接插入报告或论文中使用。

## 核心功能说明
### 1. 数据采集模块（Playwright）
- 同步模式爬取，无需复杂的异步逻辑，新手易上手
- 内置反爬策略：随机延迟（2-4秒）、页面滚动模拟、无关资源拦截
- 自动处理登录跳转与人机验证，支持手动登录后保存会话状态
- 数据自动清洗：过滤空值、转换数据类型，输出结构化Excel文件

### 2. 数据分析模块（Pandas）
- 整体/各区域二手房平均房价统计
- 热门区域房源数量与占比计算
- 房源面积段分布统计（按50㎡/80㎡/100㎡等分段）
- 数据清洗：自动处理缺失值、异常值，确保后续分析稳定

### 3. 数据可视化模块（Matplotlib）
- 区域平均房价对比柱状图，直观展示不同区域房价差异
- 热门区域房源占比饼图，快速了解市场分布情况
- 房源面积分布折线图，分析主流房源面积段
- 内置中文乱码修复配置，图表标签清晰可直接使用

## 关键配置说明
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 爬取延迟 | 2-4秒随机延迟 | 避免被链家风控识别，可根据情况调整为3-5秒 |
| 浏览器模式 | 有头模式（headless=False） | 方便调试，稳定运行后可改为无头模式提升效率 |
| 数据文件路径 | 同级目录下的`result.xlsx` | 可在代码中修改文件路径参数 |
| 超时时间 | 60秒/页 | 避免页面加载过慢导致程序报错 |

## 注意事项
1. 链家网有访问频率限制，请勿短时间内爬取大量页面，避免IP被风控
2. 遇到人机验证时，需在弹出的浏览器窗口手动完成，程序会自动继续
3. 若出现Matplotlib中文乱码，可修改`rcParams`配置更换字体
4. 爬取的二手房数据仅用于学习分析，请勿用于商业用途
